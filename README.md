# DisputeDeck

DisputeDeck is a browser-based chargeback evidence packet builder for ecommerce merchants.

Live site: https://disputedeck.netlify.app/

Products:

- DisputeDeck Pro: https://balagtas6.gumroad.com/l/disputedeck-pro
- Chargeback Packet Setup: https://balagtas6.gumroad.com/l/chargeback-packet-setup

## Netlify

Build command: leave blank.
Publish directory: `.`
Functions directory: `netlify/functions`

Environment variables for license verification:

```text
GUMROAD_PRODUCT_PERMALINK=disputedeck-pro
LICENSE_MAX_USES=10
```
